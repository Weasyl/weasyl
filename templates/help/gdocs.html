How do I find the embed link?

For documents

In the File menu of your document, there is a Publish to the web… option.

In this section, select Start publishing.

The embed link is located inside the Embed code block of HTML. Pasting just the link is not necessary; Weasyl will take the link out of the whole embed code.

For anything else

Instead of a Publish to the web… option, PDFs will have an Embed this PDF file… option. Anything else will have an Embed link… option. The block of HTML in the resulting dialog contains the embed link. As with documents, pasting just the link is not necessary; Weasyl will take the link out of the whole embed code.
