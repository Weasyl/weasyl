Using Search on Weasyl

Standard search types

building chicago
 Searches for submissions tagged with both building and chicago, with the newest submissions listed first. 
|building |architecture
 Searches for submissions tagged with either building or architecture (or both!), with the newest submissions listed first. 
|building |architecture +urban +abandoned -razed
 Searches for submissions tagged with either building or architecture or both, AND tagged with both urban and abandoned, and NOT tagged with razed. The order in which you type the search terms doesn't change the search results or the order presented (newest first). 
architecture user:JoeCool user:JayRandom
 Searches for submissions posted by either Weasyl user JoeCool or JayRandom which are tagged with architecture. 
architecture -user:AnnoyingDude
 Searches for submissions which are tagged with architecture EXCLUDING those posted by Weasyl user AnnoyingDude.

Search modifiers (can only be used with at least one search term above)

#submission
#journal
#character
